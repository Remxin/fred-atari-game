# 🎮 **Atari Fred Game** 🎮
## One of the biggest Atari XL/XE hits now in a form of **web application**.
## 📱 Now also on mobile! 
\
Page is **hosted** on: 
## 🌐 [Play the game!](http://atari-fred.remxin.ct8.pl) 🌐 

### **Launch the application on local machine:**  
- For developement mode
```
npm run dev
```
- For production mode
```
npm run prod
```
